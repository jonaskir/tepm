Text Entry Performance Measures

Custom demos can be created by adding query parameters to the URL of the site (https://jonaskir.github.io/tepm). The available query parameters are explained in the subsequent table. Click here for an example.

Query Parameters

All available query parameters are explained in the subsequent table. The default value is used if the query parameter is not set. Parameters are specified in a key=valueformat. The first query parameter is prepended with a ?, every other query parameter is prepended with a & (cf. example above).

Query Parameter Description Default Value
prompt Sets the phrase which is presented to the user for transcription. the quick brown fox jumps over the lazy dog
write Sets the text which is automatically written in the input field. Use < for backspaces, [abc] to add the characters "abc" in one action and [<<<] to backspace three characters in one action. (empty)
delay Sets the delay in milliseconds between actions. 200
hide Hides the specified measures. Measures are specified by their ID (see below), separated by a comma. (none → all measures are shown)
show Shows only the specified measures. Measures are specified by their ID (see below), separated by a comma. Empty measure categories are automatically hidden as well. Setting this value also hides the introductory words. (none) → all measures are shown
mode If set to demo, the footer is hidden. (none)
